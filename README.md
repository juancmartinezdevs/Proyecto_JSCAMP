# 🚀 Proyecto_JSCAMP

Aplicación web desarrollada con tecnologías modernas de JavaScript enfocada en la creación de interfaces dinámicas, componentes reutilizables y una experiencia de usuario responsive y moderna.

## 📌 Descripción

Este proyecto fue construido como práctica integral de desarrollo web full stack, aplicando conceptos de frontend, manejo de componentes, diseño responsive y buenas prácticas de desarrollo.

La aplicación busca ofrecer una estructura moderna y escalable utilizando herramientas actuales del ecosistema JavaScript.

---

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- React
- Vite
- Node.js
- Git & GitHub

---

## ✨ Características

- ✅ Diseño responsive
- ✅ Componentes reutilizables
- ✅ Navegación dinámica
- ✅ Interfaz moderna
- ✅ Optimización de rendimiento
- ✅ Organización modular del código
- ✅ Buenas prácticas de desarrollo
